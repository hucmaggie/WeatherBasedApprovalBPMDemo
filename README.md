WeatherBasedApprovalBPMDemo
===========================

This is the BPM Demo that provides examples of human task, timer, form designer, and custom work item handler.

1. Install BPMSuite 6.1.0.
2. Log into Business Central in BPMSuite
3. Go to Authoring -> Administration -> Repositories -> Clone Repository
4. Give it a repository name, assign it to an organizational unit, and enter the Git URL of this Github repo.

For the source of the two custom work item handlers, please see:
https://github.com/ghoelzer-rht/bpms_customWorkItemHandler
